Create the `crates/server` crate with axum HTTP server, file-level graph projection from CodeGraph, and embedded static asset serving. This plan establishes the complete Rust backend for Phase 4.

Purpose: The server crate is the bridge between the indexer's in-memory CodeGraph and the browser client. Static assets are embedded at compile time via rust-embed for single-binary distribution.

**4. Create `crates/server/src/graph_api.rs`** with these exact types and the file-level projection function:

Response types (all derive `Debug, Clone, serde::Serialize`):
- `FileNode` with fields: `id: String` (file_path), `path: String` (relative path), `filename: String` (basename or "parent/basename" for duplicates per D-54), `export_counts: ExportCounts`, `radius: f32` (per D-53: `8.0 + (total as f32 / 20.0 * 16.0).min(16.0)`), `incoming: usize`, `outgoing: usize` (edge counts for tooltip D-55)
- `ExportCounts` with fields: `functions: u32`, `classes: u32`, `types: u32`, `interfaces: u32`, `hooks: u32`, `enums: u32`, `total: u32` (Default derive)
- `FileEdge` with fields: `source: String`, `target: String` (file_path IDs)
- `ScanStats` with fields: `files: usize`, `symbols: usize`, `edges: usize`, `elapsed_ms: u64`
- `FileGraphResponse` with fields: `nodes: Vec<FileNode>`, `edges: Vec<FileEdge>`, `stats: ScanStats`, `project_name: String`

The `file_level_projection(graph: &CodeGraph, stats: ScanStats, project_name: String) -> FileGraphResponse` function:
1. Collect all unique file_paths from `graph.graph.node_weights()` into a `HashMap<String, ExportCounts>`
2. For each node: if `node.is_exported`, increment the matching `SymbolKind` counter in its file's ExportCounts. Match arms: `Function`, `Class`, `Type`, `Interface`, `Hook`, `Enum`. Skip `Module`.
3. Detect duplicate basenames: collect `basename -> Vec<file_path>`. For any basename appearing 2+ times, use `"parent_dir/basename"` as the filename. Otherwise use basename only. Truncate filename at 20 chars.
4. Iterate `graph.graph.edge_indices()`, get endpoints via `graph.graph.edge_endpoints(edge_idx)`, extract `src_file` and `tgt_file` from node file_paths. Collect into a `HashSet<(String, String)>` to deduplicate. Exclude self-edges (same file).
5. Compute incoming/outgoing counts per file: iterate the deduplicated edge set, increment outgoing for source file, incoming for target file.
6. Build `Vec<FileNode>` with computed radius, export_counts, incoming, outgoing. Build `Vec<FileEdge>` from the deduplicated set.

AppState and router:
- `AppState` struct (derive Clone): `file_graph: Arc<FileGraphResponse>`
- `create_router(state: AppState) -> Router`: routes `GET /` to `index_handler`, `GET /api/graph` to `graph_handler`, `GET /*path` to `static_handler` (from static_assets module). Use `.with_state(state)`.
- `graph_handler`: extracts `State<AppState>`, returns `Json(state.file_graph.clone())`
- `index_handler` and `static_handler`: delegate to `static_assets` module functions

Also create a `find_available_port(start: u16) -> (u16, tokio::net::TcpListener)` async function that loops from `start` trying `tokio::net::TcpListener::bind(format!("127.0.0.1:{}", port))`. On bind error, print `"Port {} in use, trying {}..."` to stderr and continue. Panic if no port found (unreachable in practice).

```rust
use axum::{
    extract::Path,
    http::{header, StatusCode},
    response::{IntoResponse, Response},
};

pub struct StaticFile<T>(pub T);

impl<T> IntoResponse for StaticFile<T>
where
    T: Into<String>,
{
    fn into_response(self) -> Response {
        let path = self.0.into();
        match ClientAssets::get(path.as_str()) {
            Some(content) => {
                let mime = mime_guess::from_path(&path).first_or_octet_stream();
                ([(header::CONTENT_TYPE, mime.as_ref())], content.data).into_response()
            }
            None => (StatusCode::NOT_FOUND, "404 Not Found").into_response(),
        }
    }
}
```

3. **Handlers with path traversal protection** (ASVS V5, T-04-01):
```rust
pub async fn static_handler(Path(path): Path<String>) -> impl IntoResponse {
    if path.contains("..") || path.starts_with('/') {
        return (StatusCode::BAD_REQUEST, "Invalid path").into_response();
    }
    StaticFile(path).into_response()
}

pub async fn index_handler() -> impl IntoResponse {
    StaticFile("index.html".to_string())
}
```
