<task type="checkpoint:human-verify" gate="blocking">
  <what-built>
Complete Phase 5 interaction layer: node expand/collapse (3 modes), zoom/pan/fit-to-screen, search (header + command palette), click-to-focus, blast radius overlay, dead code overlay, filter system (directory + symbol type + edge type + quick-filter pills), and navigation history (back/forward + breadcrumb).
  </what-built>
  <how-to-verify>
Start the server and run these verification steps:

1. **Start cgraph**: Run `cargo run -- <path-to-test-project>` (use the OversizeConnect project or any TypeScript project with multiple files).

2. **VIZN-03 — Expand/Collapse**:
   - Click a file node that has exports -> it should expand showing colored symbol children
   - Click it again -> symbols should collapse
   - Open Settings panel -> Display -> "Expand mode" dropdown
   - Switch to "Force" mode -> expanded symbols should rejoin the simulation with physics
   - Switch to "Stacked" mode -> symbols should appear as vertical list below parent
   - Switch back to "Orbital" -> symbols fan out in a ring

3. **VIZN-04 — Zoom/Pan/Fit**:
   - Scroll to zoom, drag background to pan (already working from Phase 4)
   - Click the expand-arrows button in the bottom-left corner -> graph should zoom to fit all nodes
   - Press F key -> same fit-to-screen behavior

4. **INTR-01 — Search**:
   - Type in the "Search symbols..." input -> matching nodes should glow purple, non-matches fade
   - Press Enter -> camera flies to the first match and activates focus
   - Press Escape -> search clears
   - Press Cmd+K (Mac) or Ctrl+K (Windows) -> command palette opens
   - Type a symbol name -> results list filters
   - Use arrow keys to navigate results, Enter to select -> focuses on that node
   - Click backdrop or press Escape -> palette closes

5. **INTR-02 — Focus Mode**:
   - Click any node -> it + direct neighbors at 100% opacity, everything else at ~10%
   - "Press Esc to exit focus" hint should appear
   - Press Escape or click background -> focus clears, all nodes restore

6. **INTR-03 — Blast Radius**:
   - Open Settings -> Analysis section -> toggle "Blast radius" ON
   - "Click a node to see its blast radius" prompt should appear
   - Click a node -> the node glows purple, all transitive dependents highlighted in lighter purple
   - Non-affected nodes fade to 15%
   - Toggle OFF or press Escape -> visual resets

7. **INTR-04 — Dead Code Overlay**:
   - Toggle "Dead code" ON in Analysis section
   - Confirmed dead symbols should show solid red border (3px) with 'x' badge
   - Suspicious dead symbols should show dashed red border with '?' badge
   - Stats line should show "N confirmed, N suspicious"
   - Toggle OFF -> borders and badges disappear

8. **INTR-05/06/07 — Filters**:
   - In Filters section, type a directory path -> only matching files remain visible
   - Uncheck "Function" checkbox -> function nodes fade
   - Uncheck "Import" edge checkbox -> import edges disappear
   - Click "Functions" pill in header bar -> toggles the Function checkbox off/on (synced)
   - Click "Reset" button -> all filters clear, all nodes/edges visible

9. **INTR-08 — Navigation History**:
   - Click through 3-4 different nodes (each activating focus)
   - Breadcrumb trail should appear below header showing the path
   - Click Back button (left arrow) -> returns to previous focused node
   - Click Forward button (right arrow) -> goes to next focused node
   - Click a breadcrumb item -> jumps to that node
   - Use Alt+Left/Alt+Right keyboard shortcuts

10. **No Console Errors**: Open browser DevTools (F12) -> Console tab. Check for zero JS errors during all the above interactions.
  </how-to-verify>
