Create `client/graph.js` -- the complete D3 visualization. All JS in a single file, no modules, no build step (per D-61). Structure as an IIFE or top-level async function.

**Main entry: `loadAndRender()` called on DOMContentLoaded:**

1. **Fetch data:**
```javascript
const data = await fetch('/api/graph').then(r => {
  if (!r.ok) throw new Error(`HTTP ${r.status}`);
  return r.json();
});
```
On fetch error: show `#error-state` with message "Failed to load graph data. Check that the cgraph server is running." Return early.

2. **Handle empty state:** If `data.nodes.length === 0`, show `#empty-state` with message from UI-SPEC copywriting contract: "cgraph found no source files in {project_name}. Check that the path is correct and contains .ts, .tsx, or other supported files." Set `#empty-message` text. Return early.

3. **Update header:** Set `#project-name` to `data.project_name`. Set `#stats` to `${data.stats.files} files • ${data.stats.symbols} symbols • ${data.stats.edges} edges • ${data.stats.elapsed_ms}ms` (use bullet character • per D-58).

4. **Create SVG:** Append SVG to `#graph` div. Width = `window.innerWidth`, height = `window.innerHeight - 40` (minus header). Background: none (body handles #1a1a2e).

5. **Define arrowhead marker** in SVG `<defs>`:
```javascript
svg.append("defs").append("marker")
    .attr("id", "arrowhead")
    .attr("viewBox", "-0 -5 10 10")
    .attr("refX", 13)
    .attr("refY", 0)
    .attr("orient", "auto")
    .attr("markerWidth", 6)
    .attr("markerHeight", 4)
  .append("path")
    .attr("d", "M 0,-5 L 10,0 L 0,5")
    .attr("fill", "#555555");
```

6. **Create container group for zoom:**
```javascript
const g = svg.append("g");
svg.call(d3.zoom()
    .scaleExtent([0.1, 10])
    .on("zoom", function(event) {
        g.attr("transform", event.transform);
    }));
```
Note: D3 v7 passes event as first callback parameter, NOT `d3.event` global.

7. **Pre-settle force simulation (VIZN-07):**
```javascript
const simulation = d3.forceSimulation(data.nodes)
    .force("link", d3.forceLink(data.edges).id(d => d.id).distance(80))
    .force("charge", d3.forceManyBody().strength(-120))
    .force("center", d3.forceCenter(width / 2, height / 2).strength(0.05))
    .force("collide", d3.forceCollide().radius(d => d.radius + 20))
    .stop();

simulation.tick(300);
```
Call `.stop()` before any rendering. Run `tick(300)` synchronously. Do NOT register a `"tick"` event handler.

8. **Render edges (in `g` group, BELOW nodes):**
Draw `<line>` elements from `data.edges`. Stroke: `#555555`, stroke-opacity: `0.4`, marker-end: `url(#arrowhead)`. After tick(300), set positions using adjusted endpoint to account for target node radius:
```javascript
function adjustedEndpoint(source, target, targetRadius) {
    const dx = target.x - source.x;
    const dy = target.y - source.y;
    const dist = Math.sqrt(dx * dx + dy * dy);
    if (dist === 0) return { x: target.x, y: target.y };
    return {
        x: target.x - (dx / dist) * (targetRadius + 8),
        y: target.y - (dy / dist) * (targetRadius + 8)
    };
}
```
Set x1/y1 to source.x/source.y, x2/y2 to adjusted target endpoint. The `+8` accounts for the arrowhead marker length.

9. **Render nodes (in `g` group, ABOVE edges):**
Draw `<circle>` elements from `data.nodes`. Attributes: cx = `d.x`, cy = `d.y`, r = `d.radius`, fill = `#4a9eff` (D-51 file node color), stroke = `#2a2a4e`, stroke-width = `1`. Set positions from pre-settled simulation (d.x, d.y already computed by tick(300)).

10. **Render labels (below nodes):**
Draw `<text>` elements. Attributes: x = `d.x`, y = `d.y + d.radius + 6 + 11` (6px gap below circle bottom + 11px font height), text-anchor = `middle`, fill = `#ffffff`, font-size = `11px`, pointer-events = `none`. Text content = `d.filename`.

11. **Hover tooltip (D-55):**
On node `mouseenter`: show `#tooltip`, position at `event.pageX + 12, event.pageY`. Populate:
- `.tooltip-path`: `d.path` (full relative path)
- `.tooltip-exports`: Build string from export_counts -- for each kind with count > 0, add "{N} {kind}" (e.g., "3 functions, 2 types"). Join with ", ". If total == 0, show "no exports".
- `.tooltip-edges`: `${d.incoming} incoming • ${d.outgoing} outgoing`
On node `mousemove`: update tooltip position.
On node `mouseleave`: hide tooltip (display: none).

12. **Legend panel collapse/expand:**
Click on `.legend-header` toggles `.legend-content` display (none/block). Toggle arrow character between ▾ (down) and ▴ (up).

13. **Directory halos (D-56):**
When `#halos-toggle` is checked:
- Group nodes by directory (extract directory from `d.path` using `d.path.substring(0, d.path.lastIndexOf('/'))` or "" for root files).
- For each directory group with 2+ nodes: compute convex hull via `d3.polygonHull(points)` where points are `[node.x, node.y]` for each node in group.
- Draw `<path>` elements with the hull polygon. Stroke: `#4a9eff` at 0.15 opacity, stroke-dasharray: `4 2`, fill: `#4a9eff` at 0.05 opacity.
- Add padding to hull by expanding each point outward from centroid by 30px.
When unchecked: remove all halo paths.
Default state: unchecked (halos OFF per D-56).

14. **Window resize handler:** On `window.resize`, update SVG width/height. Do NOT re-run simulation -- just resize the SVG element.

Wrap everything in:
```javascript
document.addEventListener('DOMContentLoaded', function() {
    loadAndRender();
});
async function loadAndRender() {
    // ... all the above
}
```
