**Step 5: `crates/core/src/extractor.rs`** — per D-16, D-17, D-18:
```rust
use std::path::Path;
use thiserror::Error;
use crate::model::{Language, SymbolNode, SymbolEdge};

#[derive(Debug, Error)]
pub enum ParseError {
    #[error("I/O error reading {path}: {source}")]
    Io {
        path: String,
        #[source]
        source: std::io::Error,
    },
    #[error("Parse produced ERROR nodes in {path} at line {line}")]
    PartialParse { path: String, line: u32 },
}

#[derive(Debug)]
pub struct ExtractionResult {
    pub nodes: Vec<SymbolNode>,
    pub edges: Vec<SymbolEdge>,
    pub errors: Vec<ParseError>,
}

pub trait Extractor {
    fn language(&self) -> Language;
    fn can_handle(&self, path: &Path) -> bool;
    fn extract(&self, path: &Path, source: &str) -> ExtractionResult;
}
```

Implement `crates/core/src/detect.rs` with language detection, parseability check, directory scanning, and tests.

**The module must contain these public items:**

1. **`detect_language(path: &Path) -> Option<Language>`** — per D-10, scans all extensions:
   - `.ts` -> `Language::TypeScript`
   - `.tsx` -> `Language::TypeScriptReact`
   - `.swift` -> `Language::Swift`
   - `.go` -> `Language::Go`
   - `.py` -> `Language::Python`
   - Any other extension -> `Language::Unknown(ext.to_string())`
   - No extension -> `None`

2. **`is_parseable(lang: &Language) -> bool`** — per D-11, D-12: returns `true` for TypeScript, TypeScriptReact, Swift, Go, Python; `false` for Unknown.

3. **`DetectionResult` struct** — per D-11:
   ```rust
   #[derive(Debug, Default)]
   pub struct DetectionResult {
       pub detected: Vec<(std::path::PathBuf, Language)>,
       pub parseable: Vec<(std::path::PathBuf, Language)>,
       pub skipped: Vec<(std::path::PathBuf, String)>,
   }
   ```

4. **`scan_directory(path: &Path) -> Result<DetectionResult, std::io::Error>`** — per D-10, D-11, D-19:
   - Use `walkdir::WalkDir::new(path)` to recursively walk
   - Skip directories whose name starts with `.` or equals `node_modules`, `dist`, `build`, `target` (use `entry.file_name().to_string_lossy()` for comparison -- handles non-UTF-8 safely)
   - For each file: call `detect_language`, categorize into detected/parseable/skipped
   - Parseable = detected AND `is_parseable` returns true
   - Skipped = detected AND language is `Unknown`
   - Files with no extension are silently skipped (not added to any category)
