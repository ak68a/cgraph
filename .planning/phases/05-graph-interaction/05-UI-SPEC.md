# Phase 5 — UI Design Contract

> Visual and interaction contract for the Graph Interaction phase. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none |
| Preset | not applicable |

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, chevron spacing |
| sm | 8px | Panel section padding, tooltip padding, compact gaps |
| md | 12px | Panel body horizontal padding, section header padding |
| lg | 16px | Header horizontal padding, default element spacing |
| xl | 24px | Between major UI sections |
| 2xl | 32px | Breadcrumb trail top offset from header |
| 3xl | 48px | Command palette vertical offset from top |

Exceptions: 6px used for panel-body top padding and search-wrap bottom margin (inherited from Phase 4, retained for consistency). 40px header height (inherited).

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 13px | 400 (regular) | 1.5 |
| Label | 11px | 400 (regular) | 1.4 |
| Heading | 18px | 500 (medium) | 1.2 |
| Section Header | 12px | 500 (medium) | 1.3 |

Source: Inherited from Phase 4 client/index.html. Two weights only: 400 and 500.

**Usage mapping (all interaction specs reference ONLY these 4 sizes):**
- 11px Label: symbol labels on expanded nodes, dead code badges, quick-filter pill text
- 12px Section Header: panel section titles, breadcrumb node names
- 13px Body: command palette input, search bar input, tooltip body, result list items
- 18px Heading: panel heading, empty state heading

## Interaction Specifications

### Node Expand/Collapse (D-70, D-71)

| Property | Value |
|----------|-------|
| Trigger | Click file node to expand; click again to collapse |
| Expand mode selector | Dropdown in settings panel "Display" section |
| Modes | Orbital ring, Force-integrated, Stacked list |
| Default mode | Orbital ring |
| Animation duration | 300ms ease-out for expand, 200ms ease-in for collapse |
| Symbol node size | 6px radius (smaller than file nodes which are 8-24px) |
| Symbol label | Symbol name, 11px Label size, positioned per expand mode |

### Search & Focus (D-73, D-74)

**Header search:**

| Property | Value |
|----------|-------|
| Placement | Header bar, left side after project-name, 200px wide |
| Placeholder | "Search symbols..." |
| Behavior | Live highlight on input (purple glow on matches, others fade to 12%) |
| Enter action | Fly camera to first match, activate focus mode |
| Escape action | Clear search, restore all nodes to full opacity |

**Command palette (Cmd+K / Ctrl+K):**

| Property | Value |
|----------|-------|
| Placement | Centered overlay, 480px wide, 48px from top |
| Background | #2a2a2a with border #3a3a3a, border-radius 12px |
| Backdrop | rgba(0,0,0,0.5) covering full viewport |
| Input | 13px Body size, 44px height, no border, transparent background |
| Results | Max 8 visible, 36px row height, hover #333 background |
| Selected item | #7f6df2 left border (3px) + #333 background |
| Dismiss | Escape key or click backdrop |

**Focus mode:**

| Property | Value |
|----------|-------|
| Trigger | Click any node (or select from search/palette) |
| Visual | Clicked node + direct neighbors at 100% opacity; all others at 10-12% |
| Edge treatment | Connected edges at 70% opacity with #a882ff stroke; others at 4% opacity |
| Exit | Escape key or click canvas background |

### Navigation History (D-75)

| Property | Value |
|----------|-------|
| Back/Forward buttons | Header bar, right side before stats, 28x28px icon buttons |
| Button style | #333 background, #999 icon, hover: #444 background #ddd icon |
| Disabled state | 40% opacity, cursor: default, no hover effect |
| Breadcrumb | Below header (absolute positioned), 8px below header bottom |
| Breadcrumb style | Inline items separated by ">" in #666, node names in #999, current node in #7f6df2 |
| Breadcrumb overflow | Horizontal scroll if wider than viewport - 32px padding |
| Max history depth | 50 entries (oldest dropped when exceeded) |

### Overlays & Analysis (D-76, D-77, D-78)

**Settings panel "Analysis" section (new):**

| Property | Value |
|----------|-------|
| Position | New panel section after "Filters", before "Display" |
| Section header | "Analysis" with chevron, collapsed by default |
| Dead code toggle | Label "Dead code" + toggle switch |
| Blast radius toggle | Label "Blast radius" + toggle switch |

**Dead code overlay:**

| Property | Value |
|----------|-------|
| Confirmed dead | #f87171 solid 3px border on node circle + small "x" badge (8x8px, top-right) |
| Suspicious dead | #f87171 dashed 2px border at 50% opacity + small "?" badge (8x8px, top-right) |
| Badge style | White text on colored background, 8px circle, 11px Label size |
| Non-dead-code nodes | Unchanged (no dimming) |
| Legend update | Show dead code count in panel when active |

**Blast radius mode:**

| Property | Value |
|----------|-------|
| Activation | Toggle "Blast radius" ON in panel, then click a node |
| Source node | #7f6df2 fill (solid purple) |
| Transitive dependents | #a882ff fill at 60% + connecting edges highlighted |
| Non-affected nodes | Fade to 15% opacity |
| Deactivation | Toggle OFF in panel, or Escape key |

### Filtering System (D-79, D-80)

**Panel filter controls (extending existing Filters section):**

| Control | Type | Default |
|---------|------|---------|
| File/directory | Text input with autocomplete dropdown | Empty (show all) |
| Symbol types | Checkboxes: Function, Class, Type, Hook, Enum | All checked |
| Edge types | Checkboxes: Import, Call, Type Reference | All checked |
| Orphans | Toggle switch (existing) | Checked |
| Combine logic | AND across all filter categories | — |

**Quick-filter pills (header bar):**

| Property | Value |
|----------|-------|
| Placement | Header bar, center area between search and nav buttons |
| Pill count | 4 pills maximum |
| Pills | "Functions" / "Classes" / "Imports" / "Calls" |
| Active style | #7f6df2 background, #fff text, border-radius 12px |
| Inactive style | #333 background, #999 text, border 1px solid #444, border-radius 12px |
| Size | Padding 4px 10px, 11px Label size |
| Behavior | Click toggles filter on/off; syncs with panel checkboxes |

### Zoom & Pan (VIZN-04)

| Property | Value |
|----------|-------|
| Zoom | Mouse wheel / trackpad pinch (already implemented) |
| Pan | Click-drag on background (already implemented) |
| Fit-to-screen button | Bottom-left corner, 32x32px, icon: expand-arrows |
| Button style | #333 background, #999 icon, border 1px solid #444, border-radius 8px |
| Button hover | #444 background, #ddd icon |
| Animation | 500ms ease-in-out zoom transition to fit all visible nodes with 48px padding |

### Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| Cmd+K / Ctrl+K | Open command palette |
| Escape | Exit focus mode / close command palette / clear search |
| Alt+Left / Alt+Right | Navigate history back/forward |
| F | Fit-to-screen (when no input is focused) |
